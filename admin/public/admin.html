Gateway Connection

Connect your store to the Prompt Commerce network so customers can discover your products when they search via AI like Claude.

How to get your Platform Key
1
Open the Prompt Commerce Gateway in your browser (the link was provided when you registered your store).
2
Sign in to the Gateway Admin panel.
3
Find your store in the list. Look for your store name in the Pending Verification or Verified tab.
4
Click the Issue Key button next to your store name. A long code starting with gk_ will appear.
5
Copy that code and paste it in the field below, then click Save.
Your store is connected to the gateway. Customers can now find your products via AI search. To replace the key with a new one, paste the new key below and click Save.

Connection Guide
  • Serper API Key: The AI needs a Serper.dev API key for web search. The free plan is generous and is usually enough for most stores.
  • Gemini API Key: Get your key from Google AI Studio. This is what powers the chat.
  • Anthropic API Key: For Claude, you need an Anthropic Console account.
